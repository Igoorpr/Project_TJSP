## About

Project TJSP

C# Console Project to access and query a case on the TJSP website. The project follows an architecture composed of service, interface, DTO, and a main controller class, organized into separate folders.
The application provides a console menu (CMD), allowing the user to choose an action:
- Access the original website.
- Perform a GET request to access the website and return the HTML without any search.
- Query a case number and return the HTML. 

The returned HTML after running the project will be saved in the project folder:
..\Project_TJSP\Project_TJSP\Project_TJSP\bin\Debug\net10.0\
With the following file names:
- GetWebsite.html (Website HTML without query)
- GetConsultProcessWebsite.html
